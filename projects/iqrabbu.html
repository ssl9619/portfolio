Project Overview

Iqrabbu — meaning "draw closer" in Arabic — is an interactive installation reimagining the traditional Emirati majlis, the communal seating space central to Gulf culture. Responsive sound and light invite visitors into a living, participatory version of this heritage space.

The installation responds to presence and movement, activating sensory feedback through a microcontroller-driven system programmed in Python. The goal was not to document culture but to make it participatory — letting people inhabit and activate it.

My contributions spanned the full pipeline: circuit design through soldering and wiring, microcontroller programming, and creating storyboards and flow charts that mapped the user journey and guided both technical and aesthetic implementation across the team.

Key Contributions

  • Designed installation concept mapping technology to Emirati cultural heritage
  • Contributed to circuit design through soldering and wiring of electronic components
  • Programmed microcontroller in Python for responsive sound and light logic
  • Created storyboards and flow charts mapping the full user journey
  • Guided technical and aesthetic implementation as part of 4-person team
